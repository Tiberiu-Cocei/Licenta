1) User
	1. UUID Id
	2. UUID PaymentMethodId (poate fi null)
	3. Text Email
	4. Text Username
	5. Text Password
	6. Integer WarningCount
	7. Boolean IsRiding
	8. Boolean IsBanned

2) Bicycle
	1. UUID Id
	2. UUID StationId
	3. Boolean IsDamaged
	4. Boolean IsStolen
	5. Boolean IsInWarehouse
	6. Text Model
	7. Integer LockNumber
	8. Integer ReportCount

3) Station
	1. UUID Id	
	2. Enum City
	3. Text Name
	4. Points Coordinates (X = Longitude, Y = Latitude)
	5. Integer CurrentCapacity
	6. Integer MaxCapacity
	Lista de biciclete si numarul lor se vor obtine printr-un join.

4) Transaction
	1. UUID Id
	2. UUID PaymentMethodId
	3. UUID RideHistoryId
	4. UUID UserId
	5. UUID BicycleId

5) RideHistory
	1. UUID TransactionId
	2. UUID StartStationId
	3. UUID PlannedStationId
	4. UUID FinishStationId
	5. Timestamp StartTime
	6. Timestamp PlannedTime
	7. Timestamp FinishTime
	8. Money InitialCost
	9. Money Penalty
	10. Money Discount (trigger daca finish station e diferit de planned sau timpul EndTime e depasit -> discount devine 0 si e adaugat la penalty)

6) PaymentMethod
	1. UUID Id
	2. UUID UserId
	3. Text CardNumber
	4. Date ExpiryDate
	5. Text FirstName
	6. Text LastName

7) Discount
	1. UUID Id
	2. UUID FromStationId
	3. UUID ToStationId
	4. Integer DiscountsLeft
	5. Float DiscountValue
	6. Timestamp StartTime
	7. Timestamp EndTime

8) Report
	1. UUID Id
	2. UUID UserId
	3. UUID BicycleId
	4. Enum Severity
	5. Text Description
	6. Boolean IsReviewed
	7. Boolean IsFake (daca e true user-ul primeste un warning; default e null; daca e fals si Severity = High, bicicleta va avea IsDamaged setat to true, nemaiputand fi folosita)

9) Inspection 
	1. UUID Id
	2. UUID StaffId
	3. UUID ReportId
	4. Text Description
	5. Boolean IsFake
	6. Date Date

10) Message
	1. UUID Id
	2. UUID AdminId
	3. UUID UserId
	4. Text Description
	5. Timestamp Date
	6. Boolean IsSeen

11) Staff
	1. Text CNP 
	2. Enum Position
	3. Text FirstName
	4. Text LastName
	5. Text Email
	6. Text PhoneNumber

12) Admin
	1. UUID Id
	2. Text Email
	3. Text Username
	4. Text Password
	5. Text FirstName
	6. Text LastName
	7. Boolean IsActive

13) Activity
	1. UUID StationId -> parte din PK
	2. Date Day -> parte din PK
	3. Integer HourFrom -> parte din PK
	4. Integer HourTo (HourFrom +1)
	5. Integer BicyclesTaken
	6. Integer BicyclesBrought
	7. Integer DiscountsFrom
	8. Integer DiscountsTo
	9. Boolean WasStationEmpty
	10. Boolean WasStationFull

14) PredictedActivity
	1. UUID StationId -> parte din PK
	2. Date Day -> parte din PK
	3. Integer Hour -> parte din PK
	4. Integer NumberOfBicycles

15) Settings
	1. Enum City
	2. Float BasePrice
	3. Float IntervalPrice
	4. Integer IntervalTime
	5. Boolean AreDiscountsUsed
	6. Float DiscountValue
	7. Boolean AreTransportsUsed

16) Transport
	1. UUID Id
	2. UUID StaffId
	3. Timestamp Date

17) TransportLine
	1. UUID Id
	2. UUID TransportId
	3. UUID BicycleId
	4. UUID StationFrom
	5. UUID StationTo
	6. Integer LockNumber
